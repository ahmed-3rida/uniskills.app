# إضافة صفحة سياسة الخصوصية ✅

## ما تم إنجازه

تم إنشاء صفحة سياسة الخصوصية الكاملة للتطبيق بنجاح.

## الملفات المضافة/المعدلة

### 1. ملف جديد: `privacy.html`
- صفحة سياسة الخصوصية الكاملة باللغة العربية
- تصميم متناسق مع باقي صفحات الموقع (نفس تصميم صفحة الشروط)
- تحتوي على جميع البنود المطلوبة:
  - جمع واستخدام المعلومات
  - تقنيات الذكاء الاصطناعي
  - الوصول من قبل أطراف ثالثة
  - حقوق إلغاء الاشتراك
  - سياسة الاحتفاظ بالبيانات
  - سياسة الأطفال
  - الأمان
  - معلومات الاتصال

### 2. تحديث `index.html`
- إضافة رابط سياسة الخصوصية في الفوتر
- فصل رابط شروط الخدمة عن سياسة الخصوصية

### 3. تحديث `terms.html`
- إضافة رابط سياسة الخصوصية في الفوتر للتناسق

## الروابط

- **الصفحة الرئيسية**: https://uniskills-app.vercel.app/
- **سياسة الخصوصية**: https://uniskills-app.vercel.app/privacy
- **شروط الخدمة**: https://uniskills-app.vercel.app/terms

## التطبيق

الرابط في التطبيق (`lib/screens/settings_screen.dart`) يعمل بشكل صحيح:
```dart
final url = Uri.parse('https://uniskills-app.vercel.app/privacy');
```

بفضل إعداد `cleanUrls: true` في `vercel.json`، سيتم توجيه `/privacy` تلقائياً إلى `/privacy.html`.

## الخطوات التالية

1. **رفع التحديثات إلى Vercel**:
   ```bash
   cd "Uniskills Web"
   git add .
   git commit -m "إضافة صفحة سياسة الخصوصية"
   git push
   ```

2. **التحقق من الصفحة**:
   - افتح https://uniskills-app.vercel.app/privacy
   - تأكد من ظهور المحتوى بشكل صحيح
   - اختبر الروابط في الفوتر

3. **اختبار من التطبيق**:
   - افتح التطبيق
   - اذهب إلى الإعدادات
   - اضغط على "سياسة الخصوصية"
   - تأكد من فتح الصفحة في المتصفح
