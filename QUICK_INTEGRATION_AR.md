# دليل الدمج السريع - الأقسام الجديدة

## نظرة سريعة

تم إنشاء 3 أقسام جديدة:
1. **شهادات الطلاب** (Testimonials)
2. **الامتحانات القادمة** (Upcoming Exams)  
3. **الأسئلة الشائعة** (FAQ)

## الدمج في 3 خطوات

### الخطوة 1: إضافة HTML

افتح `index.html` وأضف الأقسام في هذه الأماكن:

#### 1. بعد قسم About (حوالي السطر 307)
```html
<!-- About Section -->
...
</section>

<!-- أضف هنا: قسم Testimonials -->
<!-- انسخ من NEW_SECTIONS_IMPLEMENTATION.html -->
```

#### 2. بعد قسم Why Choose Us (حوالي السطر 340)
```html
<!-- Why Choose Us Section -->
...
</section>

<!-- أضف هنا: قسم Upcoming Exams -->
<!-- انسخ من NEW_SECTIONS_IMPLEMENTATION.html -->
```

#### 3. قبل قسم Download (حوالي السطر 350)
```html
<!-- أضف هنا: قسم FAQ -->
<!-- انسخ من NEW_SECTIONS_IMPLEMENTATION.html -->

<!-- Download Section -->
...
```

### الخطوة 2: إضافة CSS

**الطريقة الأولى (موصى بها):**
1. افتح `style.css`
2. اذهب لنهاية الملف
3. انسخ محتوى `NEW_SECTIONS_STYLES.css` بالكامل
4. الصقه في نهاية `style.css`

**الطريقة الثانية:**
أضف في `<head>` بعد `style.css`:
```html
<link rel="stylesheet" href="NEW_SECTIONS_STYLES.css">
```

### الخطوة 3: إضافة JavaScript

**الطريقة الأولى (موصى بها):**
1. افتح `script.js`
2. اذهب لنهاية الملف
3. انسخ محتوى `NEW_SECTIONS_SCRIPTS.js` بالكامل
4. الصقه في نهاية `script.js`

**الطريقة الثانية:**
أضف قبل `</body>`:
```html
<script src="NEW_SECTIONS_SCRIPTS.js"></script>
```

## التحقق من النجاح

بعد الدمج، افتح الموقع وتحقق من:
- ✅ ظهور 3 بطاقات شهادات مع نجوم

- ✅ الرسوم المتحركة عند التمرير
- ✅ التصميم يعمل على الموبايل

## التخصيص السريع

### تغيير الشهادات:
في HTML، ابحث عن `.testimonial-card` وعدّل:
- `testimonial-name`: الاسم
- `testimonial-role`: الدور
- `testimonial-text`: النص

### تغيير الأسئلة:
في HTML، ابحث عن `.faq-item` وعدّل:
- `faq-question h3`: السؤال
- `faq-answer p`: الإجابة

### تفعيل العداد التنازلي:
في JavaScript، عدّل دالة `updateCountdown()` وأضف تاريخ الإطلاق.

## المشاكل الشائعة

### المشكلة: الأقسام لا تظهر
**الحل**: تأكد من نسخ HTML بالكامل في المكان الصحيح

### المشكلة: التصميم مكسور
**الحل**: تأكد من إضافة CSS بالكامل

### المشكلة: FAQ لا يفتح
**الحل**: تأكد من إضافة JavaScript

### المشكلة: الترجمة لا تعمل
**الحل**: أضف المفاتيح في `translations` object

## الدعم

إذا واجهت أي مشكلة:
1. افتح Console في المتصفح (F12)
2. ابحث عن أخطاء JavaScript
3. تحقق من تحميل جميع الملفات
4. راجع `NEW_SECTIONS_GUIDE_AR.md` للتفاصيل الكاملة
